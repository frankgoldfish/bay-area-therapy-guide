# Bay Area Therapy Guide

Simple static landing page for a Bay Area pediatric therapy navigation concierge concept.

## What it is

A lightweight demand-test website aimed at Bay Area parents who need help navigating pediatric therapy options.

## Deployment

This site is deployed with GitHub Pages from the `main` branch.

## Files

- `index.html` — single-page landing page

## Local preview

Open `index.html` directly in a browser, or run a simple static server:

```bash
python3 -m http.server 8000
```
